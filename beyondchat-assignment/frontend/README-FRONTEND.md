ğŸ§© Component Architecture
The application logic is centralized in App.jsx for simplicity (given the assignment scope), but cleanly separated into:

State Management: useState for articles, loading, and error states.

Data Fetching: useEffect triggers an Axios call on mount.

Filtering Logic: Data is filtered client-side into originalArticles and aiArticles arrays.

Reusable UI: ArticleCard component handles the display logic for individual items, accepting props for dynamic styling (Blue vs Purple).
